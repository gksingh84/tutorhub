Google Apps Script — quick steps

  1. Open script.google.com.
  2. Click New project.
  3. Replace all code in Code.gs with the script from Tutor Hub (Copy Script).
  4. Save, then run createQuiz from the function dropdown and click Run.
  5. When Google asks for permission: if you see Google hasn’t verified this app, click Advanced, then click the blue “Go to … (unsafe)” link (it includes your project name)—not your email address. The email line opens a mailto: link (your mail app); only the Go to … (unsafe) link continues sign-in. Then click Allow. Normal for your own Apps Script; only proceed if you trust the code you pasted.
  6. Open View → Logs (or Executions) to copy your new Google Form link.
  7. Open the new form → Settings (gear) → Quizzes: set Release marks to Immediately after each submission (the script also installs a trigger that calls submitGrades).
  8. Under Respondents can see, turn on Missed questions and Correct answers (Apps Script cannot toggle these automatically).

Keyboard shortcuts

  • Ctrl + G — Generate quiz
  • Ctrl + S — Save quiz to bank
  • Ctrl + K — Question bank view
  • ? — This panel

Tutor Hub User Guide

Tutor Hub helps teachers create class content, worksheets, live quizzes, notes, and archives in one browser-based tool. Use this guide when you are setting up the tool for the first time or when you want a quick reminder.

1. Start Here

  1. Open Settings from the left menu.
  2. Open AI model & API keys and add your Gemini API key. OpenAI is optional.
  3. Choose the default class, subject, board, and quiz settings you use most often.
  4. Click Save settings. Your settings stay in this browser on this device.

2. Dashboard

  • The Dashboard is your home page.
  • Use the four cards to quickly open Create Content, Create Worksheet, Whiteboard, or Live Quiz.
  • The Do You Know? card shows useful teaching facts and ideas.

3. Create Content

  1. Open Create Content.
  2. Choose class, subject, board, chapter, and topic.
  3. Click Generate Content.
  4. Copy, download, export as PDF/Word, or save the content to Archive.

4. Create Worksheet

  1. Open Create Worksheet.
  2. Select grade, subject, topic, board, difficulty, question count, and question type.
  3. Use Paper format & extras for quiz/exam format options.
  4. Click Generate Worksheet, then download as PDF, Word, TXT, or save to Archive.
  5. For Google Forms, generate a worksheet first, then use the Google Apps Script card.

5. Live Quiz

  • Open Live Quiz to run quick question sessions with students.
  • It uses the AI settings and Live Quiz defaults from Settings.
  • You can generate a quiz, use a sample quiz, mark answers, and save the quiz to Archive.

6. Whiteboard

  • Use Whiteboard for explanations, diagrams, rough work, and quick practice.
  • It is useful during online classes or while discussing answers with students.

7. Notes and Archive

  • Notes stores class notes and reminders on this device.
  • Archive stores saved worksheets, content, and live quizzes.
  • Use the search boxes to quickly find older material by topic, class, subject, or label.

8. Tablet and Desktop Tips

  • On desktop, use the left menu to move between pages.
  • On Samsung or Apple tablets, tap the menu button in the top bar if the side menu is hidden.
  • Use the dropdown arrows on Settings and parameter cards to reduce scrolling.
  • If AI generation fails, check your API key, internet connection, and selected model.
